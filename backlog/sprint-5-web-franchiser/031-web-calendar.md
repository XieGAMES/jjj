# TASK-031: Календарь обслуживания - формирование графика

## Описание
Реализовать автоматическое формирование графика технического обслуживания.

## Критерии приемки
- [ ] Расчет даты следующего ТО: Дата последней проверки + Интервал
- [ ] Отслеживание наработки для ТА с указанным ресурсом
- [ ] Планирование ТО при достижении 80-90% лимита ресурса
- [ ] Цветовая маркировка: зеленый (плановое), желтый (< 5 дней), красный (просрочено)

## Технические требования
- Автоматический перерасчет при изменении данных
- Уведомления о приближающихся ТО
