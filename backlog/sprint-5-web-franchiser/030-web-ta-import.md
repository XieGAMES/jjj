# TASK-030: Меню ТА - импорт данных

## Описание
Реализовать функционал загрузки информации о ТА из файлов.

## Критерии приемки
- [ ] Загрузка из .xlsx файлов
- [ ] Загрузка из .csv файлов
- [ ] Проверка формата файла
- [ ] Проверка обязательных данных (минимум 6 полей)
- [ ] Проверка корректности значений
- [ ] Проверка отсутствия дублирования
- [ ] Уведомление о проблемных строках
- [ ] Успешная загрузка данных в систему

## Технические требования
- Предварительный просмотр данных
- Пошаговый мастер импорта
- Логирование результатов

## Приоритет
Высокий
