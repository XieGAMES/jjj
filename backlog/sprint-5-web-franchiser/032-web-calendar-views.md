# TASK-032: Календарь обслуживания - режимы отображения

## Описание
Реализовать различные режимы отображения календаря.

## Критерии приемки
- [ ] Режим 1: Календарь по всем ТА (по умолчанию)
- [ ] Режим 2: Календарь по одному ТА (через фильтр)
- [ ] Просмотр: неделя/месяц/год
- [ ] Годовой календарь - 12 отдельных месяцев
- [ ] При наведении - информация о ТА (модель, расположение, франчайзер)

## Технические требования
- Плавное переключение между режимами
- Адаптивное отображение для разных размеров экрана
- Быстрая навигация между периодами

## Приоритет
Высокий
