# TASK-033: График работ - автоматическое формирование заявок

## Описание
Реализовать систему автоматического формирования заявок на обслуживание.

## Критерии приемки
- [ ] Ежедневная проверка календаря ТО
- [ ] Автоматическое создание заявок со статусом "Новая"
- [ ] Интерактивный календарь расписания сотрудников
- [ ] Режимы просмотра: дневной (по часам), недельный (по дням)
- [ ] Режимы: по всем сотрудникам и по одному сотруднику

## Технические требования
- Фоновая задача для проверки календаря
- Логирование созданных заявок
- История изменения статусов

Высокий
