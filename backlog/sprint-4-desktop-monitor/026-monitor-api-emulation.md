# TASK-026: API эмуляции данных монитора

## Описание
Реализовать API для генерации эмулированных данных о состоянии ТА.

## Критерии приемки
- [ ] Эндпоинт для получения состояния связи
- [ ] Эндпоинт для получения загрузки товарами (кофе, сахар, молоко, стаканы, крышки)
- [ ] Эндпоинт для получения денежных средств
- [ ] Эндпоинт для получения статусов ТА
- [ ] Генерация реалистичных данных

## Технические требования
- Случайная генерация значений в допустимых диапазонах
- Периодическое обновление данных
- Возможность настройки частоты обновления
