# TASK-024: Монитор ТА - фильтры

## Описание
Реализовать систему фильтрации для мониторинга торговых автоматов.

## Критерии приемки
- [ ] Фильтр по общему состоянию (зеленый - работает, красный - не работает, синий - на обслуживании)
- [ ] Фильтр по типу подключения
- [ ] Фильтр по дополнительным статусам
- [ ] Кнопка "Применить" для активации фильтров
- [ ] Сообщение при отсутствии результатов

## Технические требования
- Множественный выбор для каждого фильтра
- Сохранение состояния фильтров
- Быстрый сброс всех фильтров

## Приоритет
Высокий
