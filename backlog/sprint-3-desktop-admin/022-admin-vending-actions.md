# TASK-022: Действия с ТА

## Описание
Реализовать действия редактирования, удаления и отвязки модема для ТА.

## Критерии приемки
- [ ] Кнопка редактирования - открытие формы редактирования
- [ ] Кнопка удаления с диалогом подтверждения
- [ ] Кнопка отвязки модема с подтверждением
- [ ] После отвязки поле модема принимает значение -1

## Технические требования
- Модальные диалоги для подтверждения
- Уведомления об успешном выполнении
- Обновление списка после действия
