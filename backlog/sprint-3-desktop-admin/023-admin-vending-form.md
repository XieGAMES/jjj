# TASK-023: Форма добавления/редактирования ТА

## Описание
Реализовать форму для добавления и редактирования информации о торговом автомате.

## Критерии приемки
- [ ] Все поля согласно макету
- [ ] Валидация всех ограничений на клиенте
- [ ] Сообщения об ошибках (дублирование серийного/инвентарного номера)
- [ ] Валидация дат (ввод в эксплуатацию, поверки, инвентаризации)
- [ ] Выпадающие списки для статуса и страны производства
- [ ] Сохранение и отмена

## Технические требования
- Использование макета из задания
- Мгновенная валидация при вводе
- Блокировка кнопки сохранения при ошибках
