# TASK-021: Группировка ТА по папкам

## Описание
Реализовать возможность группировки торговых автоматов по папкам (франчайзи).

## Критерии приемки
- [ ] Древовидная структура папок по франчайзи
- [ ] Перемещение ТА между папками
- [ ] Фильтрация по выбранной папке
- [ ] Счетчик ТА в каждой папке

## Технические требования
- Drag-and-drop для перемещения
- Сохранение структуры в БД
- Возможность создания новых папок

Средний
