# TASK-018: Список торговых автоматов

## Описание
Реализовать отображение списка всех ТА в системе.

## Критерии приемки
- [ ] Таблица с колонками: ID, Название, Модель, Компания, Модем, Адрес, В работе с, Действия
- [ ] Переключение между режимами: плитка и таблица
- [ ] По умолчанию - табличный вид
- [ ] Выделение цветом нечетных строк

## Технические требования
- Загрузка данных из API
- Кэширование для быстрого отображения
- Плавное переключение между режимами
