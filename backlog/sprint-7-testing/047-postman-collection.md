# TASK-047: Postman коллекция для API

## Описание
Создать и настроить коллекцию Postman для работы с REST API.

## Критерии приемки
- [ ] Папки для группировки: Торговые автоматы, Компании, Мониторинг
- [ ] Переменные для базового URL, ID ТА, ID компании
- [ ] Методы авторизации: login, refresh-token, logout
- [ ] Тесты для проверки статуса ответа
- [ ] Проверка структуры ответа (name, location, status)
- [ ] Описание для каждого запроса
- [ ] Экспорт коллекции в JSON

## Технические требования
- Понятные имена запросов и папок
- Корректное применение переменных
- Полезные и содержательные тесты
