# TASK-042: Мобильное приложение - протокол осмотра

## Описание
Реализовать экран протокола осмотра с чек-листом.

## Критерии приемки
- [ ] Заголовок: место размещения, модель ТА, дата, ФИО сотрудника
- [ ] Чек-лист для планового ТО:
  - Проверка общего состояния
  - Оценка уровня износа
  - Замеры рабочих параметров (давление, температура)
  - Показания счетчика (целое число до 65000)
  - Профилактические мероприятия
  - Минимум 4 фото оборудования
- [ ] Чек-лист для аварийного обслуживания:
  - Описание аварийной ситуации
  - Фото оборудования
  - Анализ причин поломки
  - Список выполненных мер
  - Рекомендации по предотвращению
  - Инструкции по дальнейшей эксплуатации

## Технические требования
- Динамический чек-лист в зависимости от типа обслуживания
- Валидация обязательных полей

## Приоритет
Высокий
