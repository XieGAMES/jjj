# TASK-044: Мобильное приложение - создание заметок

## Описание
Реализовать функционал создания и редактирования заметок.

## Критерии приемки
- [ ] Текстовый редактор: заголовок + тело заметки
- [ ] Форматирование (жирный, списки)
- [ ] Вставка фото/видео из галереи
- [ ] Голосовой ввод
- [ ] Кнопки "Сохранить" и "Прикрепить фото"
- [ ] Простой редактор (аналог Google Keep)

## Технические требования
- Локальное сохранение заметок
- Синхронизация с сервером
- Поддержка офлайн-режима

Средний
