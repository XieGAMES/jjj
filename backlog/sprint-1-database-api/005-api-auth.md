# TASK-005: Реализация аутентификации JWT

## Описание
Реализовать систему аутентификации с использованием JWT токенов.

## Критерии приемки
- [ ] Эндпоинт POST /auth/login - авторизация
- [ ] Эндпоинт POST /auth/refresh-token - обновление токена
- [ ] Эндпоинт POST /auth/logout - инвалидация токена
- [ ] Хеширование паролей
- [ ] Защита эндпоинтов

## Технические требования
- Access token с коротким временем жизни
- Refresh token с длинным временем жизни
- Безопасное хранение токенов
- Валидация токенов при каждом запросе
