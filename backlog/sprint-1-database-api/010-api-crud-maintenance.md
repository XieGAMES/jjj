# TASK-010: CRUD API для обслуживания

## Описание
Реализовать REST API для управления записями об обслуживании ТА.

## Критерии приемки
- [ ] GET /api/maintenance - список записей обслуживания
- [ ] GET /api/maintenance/{id} - детали записи
- [ ] POST /api/maintenance - создание записи
- [ ] PUT /api/maintenance/{id} - обновление записи
- [ ] DELETE /api/maintenance/{id} - удаление записи
- [ ] Фильтрация по аппарату, исполнителю, дате
- [ ] История обслуживания для конкретного ТА

## Приоритет
Средний
