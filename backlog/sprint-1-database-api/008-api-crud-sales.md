# TASK-008: CRUD API для продаж

## Описание
Реализовать REST API для управления записями о продажах.

## Критерии приемки
- [ ] GET /api/sales - список продаж с фильтрами
- [ ] GET /api/sales/{id} - детали продажи
- [ ] POST /api/sales - регистрация продажи
- [ ] Фильтрация по дате, аппарату, товару, методу оплаты
- [ ] Агрегированные данные (сумма за период)

## Приоритет
Средний
