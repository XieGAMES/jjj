# TASK-014: Профиль пользователя

## Описание
Реализовать отображение данных сотрудника и возможность просмотра профиля.

## Критерии приемки
- [ ] Отображение фото, ФИО, инициалов и роли
- [ ] Dropdown меню с опцией просмотра профиля
- [ ] Страница профиля с детальной информацией
- [ ] Возможность выхода из аккаунта

## Технические требования
- Аватар пользователя или заглушка
- Кэширование данных профиля
- Dropdown закрывается по клику вне области
