# TASK-015: Главная страница - блоки информации

## Описание
Реализовать главную страницу с 5 информационными блоками.

## Критерии приемки
- [ ] Блок "Эффективность сети" - процент работающих автоматов
- [ ] Блок "Состояние сети" - диаграмма статусов (работает/не работает/на обслуживании)
- [ ] Блок "Сводка" - данные по продажам, инкассации, обслуживании
- [ ] Блок "Динамика продаж за 10 дней" - график с фильтрацией по сумме/количеству
- [ ] Блок "Новости" - информация с новостями

## Технические требования
- Адаптивное расположение блоков
- Обновление данных из API
- Визуально привлекательные графики

## Приоритет
Высокий
