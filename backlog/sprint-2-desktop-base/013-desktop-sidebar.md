# TASK-013: Боковое меню (Sidebar)

## Описание
Реализовать боковую панель навигации с возможностью сворачивания.

## Критерии приемки
- [ ] Боковое меню с пунктами: Главная, Монитор ТА, Детальные отчеты, Учет ТМЦ, Администрирование
- [ ] Вложенные пункты меню с раскрытием/скрытием
- [ ] Минималистичный дизайн
- [ ] Сворачивание sidebar для оптимизации пространства
- [ ] Выделение активного пункта меню

## Технические требования
- Плавная анимация сворачивания
- Сохранение состояния меню между сессиями
- Иконки для пунктов меню
