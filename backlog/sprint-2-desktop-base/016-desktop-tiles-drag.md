# TASK-016: Перетаскивание и скрытие плиток

## Описание
Реализовать возможность кастомизации расположения блоков на главной странице.

## Критерии приемки
- [ ] Drag-and-drop для перемещения плиток
- [ ] Возможность скрытия отдельных плиток
- [ ] Сохранение расположения между сессиями
- [ ] Кнопка сброса к расположению по умолчанию

## Технические требования
- Плавная анимация перетаскивания
- Визуальная индикация места для размещения
- Сохранение настроек в локальное хранилище
